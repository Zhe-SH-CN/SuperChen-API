SuperChen API 接口文档

接口名称：图片API

介绍：随机或者指定获取图片

接口地址：域名/apis/getimages/id/token

调用方法:

输入id为 -1时 随机返回图片地址

传入大于 0 的id返回 指定图片

返回结果:

例： { "code": 200, "msg": "https://w.wallhaven.cc/full/y8/wallhaven-y8vlyk.jpg", "date": "2022-43-19" }

code --- 状态码 200 代表调用成功 403 拒绝访问

msg --- 响应的数据 图片的地址

data --- 日期
