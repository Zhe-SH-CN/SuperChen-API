# SuperChen_API 平台

#### 介绍
这是一个提供API 的平台





#### 软件架构
软件架构说明:

后端
Mysql
redis
springboot + mybatis-plus+thymeleaf
nginx

前端
bootstrap
jquery
ajax



#### 安装教程

1.  准备一台服务器
2.  配置java环境
3.  安装Mysql
4.  安装redis
5.  安装nginx
#### 使用说明

1.  修改配置文件中的mysql数据库配置
2.  修改配置文件中的redis数据库配置
3.  修改配置文件中的邮件配置
4.  配置nginx的动静分离和反向代理(8080端口)
